Obsession Girls Project

Skills Used

  • ASP.NET
  • ASP Membership
  • Ajax
  • HTML
  • CSS
  • JavaScript
  • SQL

Summary

This was my first project and one of the ones I am the most proud of. It is also the project that made me realise how much I love code! I worked with my Dad on this, and learned a lot very quickly. The site used ASP Membership to control which pages could be accessed by which members. Information was stored in an SQL database and pulled in as and when needed.

We created our own groups/forums, user areas, comment system and messaging system in ASP.

The content is styled with HTML and CSS entirely by me.


Notes
